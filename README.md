## Installation

The preferred way to install the library is through [composer](https://getcomposer.org/download/).

Either run
```
php composer.phar require --prefer-dist weblement/yii2
```

or add
```json
{
    "require": {
        "weblement/yii2": "~1.0.0"
    }
}
```
to your `composer.json` file.
